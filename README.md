# TODO APPLICATION

## Setup

The first thing to do is to clone the repository:

```sh
$ git clone https://github.com/ramsey009/todof.git
$ cd todof
```

```sh
`npm i` to install dependency
```

```
And navigate to `http://localhost:3000/`.
```
